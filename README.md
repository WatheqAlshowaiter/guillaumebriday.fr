# guillaumebriday.fr

Source code of [guillaumebriday.fr](https://guillaumebriday.fr) built with [Gatsby](https://www.gatsbyjs.org) and [Tailwind CSS](https://tailwindcss.com/).

## Running in development

```bash
$ yarn
$ yarn develop
```

Format code:
```bash
$ yarn format
```
